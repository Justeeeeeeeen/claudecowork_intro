Getting Started with Claude Cowork
Everything you need to set up before the session

Claude Cowork is a desktop AI agent that works directly with your local files — documents, spreadsheets, presentations — from inside the Claude Desktop app. Unlike the browser version of Claude, Cowork can open, read, edit, and create files on your machine. No coding skills needed.

What You Need to Install
🖥️
Claude Desktop App

The main app that runs Cowork. Download, install, sign in, then click the Cowork tab in the sidebar. On first run it downloads a small VM image — just let it complete.

↓ claude.ai/download
🐍
Python

Powers scripts and automations that run under the hood. You don't need to write any code — it just needs to be installed on your machine.

↓ python.org/downloads Version 3.11 or newer
⚠️ During installation, tick "Add Python to PATH" — easy to miss
⬡
Node.js

Required by some Cowork plugins and connectors. Not needed immediately but good to have for when you want to extend Cowork's capabilities.

↓ nodejs.org Download the LTS version
🔌
MCP Connectors

How Cowork connects to Gmail, Google Calendar, Google Drive, and other services. Once linked, Cowork can read emails, check your calendar, and pull data — all without switching tabs. One-click authorisation per service, no coding needed.

⚙️ Claude Desktop → Settings → Connectors
Setup Checklist
Claude Desktop installed and signed in
Cowork tab visible in the sidebar
Folder access granted in Cowork settings
Python installed and added to PATH
Node.js installed
At least one MCP connector linked (e.g. Gmail)
Quick Tips
Start with one folder Grant Cowork access to a single folder first, read-only if possible. You can always add more later.
You're always in control Cowork shows you what it's doing and asks before making any changes to your files.
Best first task Pick something repetitive you do manually every week — formatting a report, summarising a file, drafting an email.
No coding needed Describe what you want in plain language and Cowork figures out the rest.
